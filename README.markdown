wwwrefe.vim
===========

reference tool of [rurema](http://doc.okkez.net/).

usage
-----

command line

    :WWWRefe Class#method

unite.vim

    :Unite wwwrefe

variables
---------

g:wwwrefe_myrurema
    
    use myrurema.
    default : 0

    install
      gem install myrurema
      rurema --init
